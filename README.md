# programacionFullstack-UTU-2026

Este repositorio fue creado para organizar y almacenar el material trabajado durante el año 2026 en las **Materias Respectvas** en UTU.

## Objetivo del repositorio

El objetivo de este repositorio es mantener un registro ordenado de las actividades, ejercicios, apuntes y proyectos desarrollados durante el curso.
Además, permite seguir el progreso a lo largo del año y centralizar todo el material en un solo lugar.

## Organización del repositorio

El repositorio está organizado en **10 carpetas principales**:

* **9 carpetas correspondientes a los meses de clase**, desde marzo hasta noviembre.
* **1 carpeta adicional llamada `proyecto`**, destinada al proyecto principal del curso.

Esta estructura permite organizar el contenido de forma cronológica, facilitando encontrar el material trabajado en cada período del año.

## Estructura de carpetas

```
programacionFullstack-UTU-2026
│
├── marzo
├── abril
├── mayo
├── junio
├── julio
├── agosto
├── septiembre
├── octubre
├── noviembre
│
└── proyecto
```

### Carpetas mensuales

Cada carpeta mensual se utiliza para guardar el material correspondiente a las clases de ese mes.
Dentro de estas carpetas pueden encontrarse:

* ejercicios
* apuntes
* ejemplos de código
* tareas o actividades prácticas

### Carpeta `proyecto`

La carpeta `proyecto` está destinada al desarrollo del **proyecto principal del curso**, donde se aplicarán los conocimientos adquiridos durante el año.

## Uso del repositorio

Este repositorio funciona como un espacio de organización y seguimiento del aprendizaje durante el curso de Programación Fullstack.

Permite:

* mantener el contenido ordenado por mes
* acceder fácilmente al material trabajado
* documentar el progreso a lo largo del año
