HTML

HTML (HyperText Markup Language) es el lenguaje estándar de marcado para la creación de páginas web. Junto a CSS y JavaScript, HTML es una de las tres tecnologías fundamentales de la Web.

Historia

Tim Berners-Lee describió HTML en un documento llamado "HTML Tags" a finales de 1991. Las primeras versiones de HTML tenían pocas etiquetas, orientadas principalmente a documentos académicos. Con la llegada de HTML 4 y más tarde HTML5, el lenguaje evolucionó para soportar multimedia, formularios complejos y accesibilidad.

HTML5

HTML5 fue publicado en octubre de 2014 por el W3C. Introdujo nuevas etiquetas semánticas como <article>, <section>, <nav> y <aside>, así como soporte nativo para audio y video.

Marcado

HTML utiliza "etiquetas" para marcar el contenido. Las etiquetas son palabras clave rodeadas por corchetes angulares. Las etiquetas generalmente vienen en pares (apertura y cierre).

Elementos principales

  • <html> — raíz del documento
  • <head> — metadatos del documento
  • <body> — contenido visible
  • <h1>–<h6> — encabezados
  • <p> — párrafo
  • <a> — enlace
  • <img> — imagen
  • <ul>, <ol>, <li> — listas
  • <table> — tabla
  • <form> — formulario
