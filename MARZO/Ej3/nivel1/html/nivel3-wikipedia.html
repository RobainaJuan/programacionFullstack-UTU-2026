HTML

✔

HTML (HyperText Markup Language, 'lenguaje de marcas de hipertexto') es el lenguaje de marcado estándar para la creación de páginas web. Junto a CSS y JavaScript, HTML es una de las tres tecnologías fundamentales de la World Wide Web.

Historia [editar]

Tim Berners-Lee describió HTML en un documento llamado "HTML Tags" a finales de 1991. Este primer borrador definía 18 elementos basados en SGML.

HTML5 [editar]

HTML5 fue publicado como recomendación del W3C el 28 de octubre de 2014. Introdujo nuevas etiquetas semánticas y soporte nativo para audio y video.

Marcado [editar]

El marcado HTML consiste en etiquetas delimitadas por corchetes angulares (< y >). La mayoría de los elementos requieren una etiqueta de apertura y una de cierre.

Elementos principales [editar]

  • <html> — raíz del documento
  • <head> — metadatos del documento
  • <body> — contenido visible
  • <p> — párrafo
  • <a> — enlace (hipervínculo)
  • <img> — imagen
